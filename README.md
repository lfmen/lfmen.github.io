## Arquitectura

El desarrollo del sitio se realizó utilizando exclusivamente **HTML5 y CSS3 Vanilla**, prescindiendo de librerías externas o frameworks de JavaScript. Esta decisión arquitectónica se fundamenta en los siguientes criterios técnicos:

- **Rendimiento:** La ausencia de *bundles* de JavaScript optimiza el tiempo de carga y reduce significativamente el *First Contentful Paint*.
- **Mantenibilidad:** El código fuente opera de forma independiente y evita riesgos de obsolescencia asociados a la deprecación de dependencias o actualizaciones de frameworks.
- **Control estructural:** Los componentes visuales e interactivos (como el Dark Mode y las animaciones CSS) fueron desarrollados de forma nativa.
